# NodeDrop — Starter Repo

Welcome to NodeDrop! 💬

This is the starter template for the NodeDrop workshop.
You will build a real-time chat app from scratch during the session.

## Setup

1. Clone this repo
2. Run `npm install`
3. Run `node index.js`
4. Open `http://localhost:3000`

## Stack
- Node.js
- Express
- Socket.io
- Plain HTML / CSS / JS
